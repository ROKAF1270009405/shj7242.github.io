> Hello world 프린트

~~~

#include <stdio.h>
// stdio.h 를 인클루드하는 부분 여기에 printf 뿐만아니라 여러 함수를 포함한다.

int main(void)

{
	printf("Hello, world.");

	return 0;
}

~~~

이클립스를 설치하고 첫번째 관문인 hello world 를 찍어보았다. 첫 단추를 잘 꿰어낸 느낌이 든다.
